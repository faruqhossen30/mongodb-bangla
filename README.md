#### MySQL vs MongoDB Database Term
---

| My SQL  | MongoDB  |
| ------------ | ------------ |
| Database  | Database  |
| Table  | Collection  |
| Index  | Index  |
| Row  | BSON Document  |
| Colum  | BSON Field  |
| Join  | Embedded Document and Linking  |
| Primary Key  | Primary Key  |
| Group by  | Aggregation  |
